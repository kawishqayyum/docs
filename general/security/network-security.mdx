title: Bright Data 如何确保网络安全？
sidebarTitle: 网络安全 
---
大多数经过常规互联网服务提供商和虚拟互联网服务提供商的流量都使用 SSL 隧道进行加密，因此互联网服务提供商看不到其内容。 因此，互联网服务提供商可用的保护网络安全和防止非法活动的方法是有限的，并且主要针对大规模 DDoS 预防和域名黑名单。 但是，在 Bright Data，我们做得更多：  

| 元素 | Bright Data 虚拟互联网服务提供商 | 常规互联网服务提供商 |
| --- | --- | --- |
| 需要注册 | 是 | 是 |
| 需要付款方式 | 是 | 是 |
| 举报滥用行为的方法和程序 | 是 | 是 |
| 在线 DDoS 检测 | 是 | 是 |
| 全面屏蔽最热门的 DDoS 方法<sup>[2](#myfootnote2)</sup> | 是 | 否 |
| 每位客户的“了解客户”程序 | 是 | 否 |
| 如果客户开始定位“了解客户”期间未报告的域名，则会发出合规警报 | 是 | 否 |
| 针对广告欺诈、虚假评论、虚假点赞、虚假账户创建、垃圾邮件等，部署主动屏蔽 | 是<sup>[1](#myfootnote1)</sup> | 否 |
| 阻止服务级别滥用或使用非网络流量攻击 <sup>[3](#myfootnote3)</sup> | 是 | 否 |
  
---

> **底线**  
> Bright Data 虚拟互联网服务提供商网络比常规互联网服务提供商更安全，并且被用于攻击和滥用的可能性更低。

<Warning>
Bright Data 不允许使用其产品和系统导致任何网络资源无法供其预期用户使用，包括但不限于通过拒绝服务（DoS）或分布式拒绝服务（DDoS）攻击
</Warning>
