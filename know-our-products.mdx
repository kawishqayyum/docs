Bright Data 为每个用例提供一系列代理和抓取自动化解决方案，使您可以专注于数据收集，其余工作交由我们的技术负责。

请在下方查看我们的代理产品，了解如何精确地集成业界最先进的代理产品和基础架构，以满足您的需求。

## 产品 #1: 解锁抓取解决方案
让我们为您管理完整的代理和解锁基础架构。

<Card>
**单步抓取** - 您将 URL 传递给我们，而我们负责全面的解锁、验证码、指纹识别等，并返回给您一个干净的页面(HTML、JSON 等):
<CardGroup>
  <Card title="Web Unlocker" icon="unlock" href="https://docs.brightdata.com/scraping-automation/web-unlocker/introduction">
  在不被屏蔽的情况下使用 HTTP/s 请求从任何网站抓取数据
  </Card>
  <Card title="SERP API" icon="magnifying-glass" href="https://docs.brightdata.com/scraping-automation/serp-api/introduction">
  F便于搜索引擎抓取。支持 Google、Bing、雅虎、DuckDuckGo、Yandex、百度和 Naver 搜索 API
  </Card>
</CardGroup>
<br></br>
**多步抓取** - 通过我们的 API 启动无限制的云托管浏览器会话，同时我们会为您处理完整的解锁基础架构，包括代理、验证码、重试和指纹识别:
<Card title="Scraping Browser" icon="window-restore" href="https://docs.brightdata.com/scraping-automation/scraping-browser/introduction">
通过创建浏览器会话并导航到所需的确切数据，从任何网站抓取数据。支持 `puppeteer`、`playwright` 和 `selenium`.
</Card>
</Card>
