---
title: 请求收集
---

**端点:** `POST` `/datasets/request_collection`

第一步是确定要收集哪些数据。 在指定新数据集收集参数（如定义数据集 ID 和类型）或发现新数据或指定 URL 收集时使用。

## 查询参数
<ParamField query="dataset_id" type="string" required>
数据集 ID
</ParamField>

<ParamField query="type" type="string" required>
|值|描述|
|-|-|
|`discover_new`| |
|`url_collection`| |
</ParamField>

## `POST` body
<ParamField body="inputs" type="array" required>
Array - json
```json 正文示例
[
  {
    "url":"https://www.linkedin.com/company/sendle/"
  },
  {
    "url":"https://www.linkedin.com/company/stallionexpress/"
  },
  {
    "url":"https://www.linkedin.com/company/595711"
  }
]
```
</ParamField>

<ParamField body="file" type="array" required>
multipart - csv
</ParamField>

<RequestExample>
```sh
curl -k "https://api.brightdata.com/datasets/request_collection?dataset_id=gd_l1vikfnt1wgvvqz95w&type=url_collection" -H "Authorization: Bearer API_TOKEN" -H "Content-Type: application/json" -d '[{"id":"user-id"}]'
```
</RequestExample>

<ResponseExample>
```json
{
    "request_id": "j_lsu5sdup1ggkc5umby"
}
```
</ResponseExample>
