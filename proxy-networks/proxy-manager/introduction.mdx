Bright Data Proxy Manager是一款应用程序，让您可以通过我们的应用程序界面或API控制、管理和监控所有代理。

这款应用程序是由开发人员为开发人员构建的高级开源应用程序工具 [(GitHub)](https://github.com/luminati-io/luminati-proxy)，允许独特的代理自定义、规则实施和外部代理集成。


最适用于：
--------

* 自动处理不同代理产品和IP之间的流量分配
* 使用屏蔽某些类型数据的规则来减少带宽
* 监控HAR日志，可实时预览整个流量
* 使用白名单或生成的令牌来管理用户访问
* 希望自定义和管理其代理流程和基础设施的团队
