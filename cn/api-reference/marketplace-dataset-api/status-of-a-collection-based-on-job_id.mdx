---
title: 按 job_id 排序的收集状态
---

**端点:** `GET` `/datasets/request_collection`

这一步是为了查询现有数据收集请求的信息，以了解其当前状态。它用于检查收集请求的状态或详细信息，如总行数、数据的新鲜度或其他相关详细信息。

## 查询参数
<ParamField query="request_id" type="string" required>
从先前的 API 获取。
</ParamField>

<ParamField query="freshness_ms" type="string">
设置数据新鲜度。
</ParamField>

<RequestExample>
```sh
curl -k "https://api.brightdata.com/datasets/request_collection?request_id=j_lsu5sdup1ggkc5umby" -H "Authorization: Bearer API_TOKEN"
```
</RequestExample>
