Web Unlocker是我们的代理解锁解决方案之一，旨在由我们为您全面处理代理工作和基础设施解锁，让您可以专注于数据收集。

您只需向目标网站发送一个代理请求，我们就会将干净的HTML和/或JSON数据交付给您 。就是这么简单！

在幕后，我们的Web Unlocker智能算法会处理整个不断变化的流程，包括寻找最佳代理网络、自定义标题、指纹识别、CAPTCHA验证码等。

## 最适用于
+ 从任何网站抓取数据，不会被屏蔽
+ 模拟真实用户的网络行为，对使用复杂检测方法的网站也能进行抓取
+ 内部没有可扩展的解锁基础设施的团队
+ 只需为成功的请求付费

<Note>
Web Unlocker不适用于浏览器或Adspower、Puppeteer、Playwright、Multilogin（MLA）等第三方工具。如果您需要与浏览器交互并使用此类工具的解决方案，请查看我们的Scraping Browser解决方案，或查看我们其他可以为您提供完整数据集的网页数据产品 。
</Note>

<Tip>
如果您在寻找专门用于搜索引擎的解锁解决方案，请查看我们的[SERP API](scraping-automation/serp-api)
</Tip>
