SERP API是我们的代理解锁解决方案之一，旨在由我们为您全面处理代理工作、解锁和基础架构解析，让您可以专注于数据收集。

您只需向特定的目標搜索引擎发送一个代理请求，我们就会将已解析（或未解析）的HTML/JSON干净数据交付给您。就是这么简单。

在幕后，我们SERP API的强大平台管理着整个抓取和解锁流程（从选择最佳代理网络到自定义标头、指纹识别、CAPTCHA验证码破解），最后返回干浄的结构化HTML或已解析的JSON数据:  
http://docs.brightdata.com/scraping-automation/serp-api/parsing-search-results

### Best for
* 抓取搜索引擎，例如Google、Bing、DuckDuckGo、Yandex、百度、Yahoo和Naver 搜索API
* 自然关键词跟踪、品牌保护、价格比较、市场研究、版权侵权检测、广告情报
* 发出多个并发请求
