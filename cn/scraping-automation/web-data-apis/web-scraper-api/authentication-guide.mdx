---
title: 身份验证指南 
---

Bright Data API 使用 “API 令牌” 对请求进行身份验证。 您可以在控制面板的 “账户设置” 下查看和管理您的 API 令牌。

## 什么是 Bright Data 的 API 令牌？
Bright Data 的 API 令牌是通过 API 向 Bright Data 发送请求时替换您的证书（即电子邮件地址和密码）的一种安全方法。

<Tip>
[指南：如何生成 API 令牌？](https://docs.brightdata.com/cn/api-reference/unlocker/api_token#api-token)
</Tip>

<Note>
只有账户管理员才能为账户用户生成新令牌
</Note>
