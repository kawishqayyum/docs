---
title: Bright Data's API 令牌
---

Bright Data API 令牌是一种在通过 API 向 Bright Data 发送请求时替换您的证书（即电子邮件地址和密码）的安全方法。使用 API 需要 API 令牌，这样可以始终确保您的账户凭证安全可靠。

## 生成 API 令牌

<Steps>

<Step title="登录">
登录到 Bright Data 的控制面板。导航到您的账户设置
<Frame>
![account-settings.png](/images/api-reference/unlocker/api_token/account-settings.png)
</Frame>
</Step>

<Step title="点击添加令牌">
<Warning>
If the 'API tokens' section is not visible, switch to your **admin account**. You need to be logged in as admin to generate an API token. 
</Warning>
<Frame>
![add-token.png](/images/api-reference/unlocker/api_token/add-token.png)
</Frame>
</Step>

<Step title="设置令牌到期日期">
设置令牌到期日期（或选择 “无限制”）、许可权限，选择所要生成令牌的用户，然后点击 “保存”
<Frame>
![save-token.png](/images/api-reference/unlocker/api_token/save-token.png)
</Frame>
</Step>

<Step title="保存您的令牌">
<Warning>
您的令牌生成之后，务必将其保存，因其仅供一次性使用（仅限查看！）  
</Warning>

<Frame>
![new-api-token.png](/images/api-reference/unlocker/api_token/new-api-token.png)
</Frame>
</Step>
  
</Steps>

<Tip>
创建令牌之后，您将无法再次查看，但是，您仍然可以在任何时候对其刷新、更改其到期日期以及将其撤消。
<Frame>
![token-edit-buttons.png](/images/api-reference/unlocker/api_token/token-edit-buttons.png)
</Frame>
</Tip>
