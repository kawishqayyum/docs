## 异步请求
请求在后台无缝处理，允许您在稍后更方便的时间通过指定端点收集响应，从而提高稳定性、灵活性和效率。
[Learn more](/scraping-automation/serp-api/asynchronous-requests)

<Tip>
### 将我们的 webhook IP 列入白名单

我们的异步 webhook 交付会从一对稳定的 IP 地址发送通知：
1. `100.27.150.189`
2. `18.214.10.85`

客户可能需要在其一侧将这些 IP 列入白名单。
</Tip>
